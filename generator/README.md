This is where we generate html files at [..](..) (except for `index.html`) from
files at [../raw/](../raw/)

To install dependencies, run `git submodule update --init`.

To generate:

  * Edit files at [../raw/](../raw/) (if you add new ones, also edit [../truthmap.js](../truthmap.js)).
  * run `./generate.sh`.
